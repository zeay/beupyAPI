Messenger Feature

Code snippets at right perform sending message feature for your website .
To integrate just follow the same procedure all the fields and name are mandatory and schould be same as shown you can add a optional field for subject with name = "subject" only.

Example form for messenger

<form action="http://apibeupy.openode.io/messenger/your@email.com" method="GET">
  <input type="text" name="userName">
  <input type="email" name="email">
  <input type="text" name="message">
  <input type="submit" value="Send">
</form>

Getting IP

Do Ajax call to our endpoint to get ip of your visited user . Do whatever you want to do with the result, memorise the url.
"http://apibeupy.openode.io/getIp"

Example form for IP service

<script>
  var xmlhttp = new XMLHttpRequest();
  var domain = "https://apibeupy.openode.io/getIp";
  xmlhttp.onreadystatechange = function(){
let result = JSON.parse(this.response);
console.log(result);
  }
  xmlhttp.open('GET', domain, true);
  xmlhttp.send();
</script>
